## 2. 핵심기능

1. **JWT를 이용한 로그인과 회원 가입 구현**

2. **로그인 사용자에 따른 댓글 작성, 포스트 작성 그리고 삭제 권한 기능**

3. **개인 프로필 이미지 저장 및 상태 설명 기능** 

4. **페이저를 통한 포스트 뷰의 사용자 편의성** 

5. **좋아요와 댓글의 수를 직관적으로 볼 수 있도록 표시** 

6. **파일 저장 중복 방지와 보안 유지를 위한 랜덤 값 이름 부여 기능**
